# Vendor Performance Analysis (SQL + Python + Power BI)

End-to-end vendor analytics project that ingests raw inventory + purchasing data, builds vendor-level KPIs, and delivers insights through **Python analysis (EDA + modeling-style insights)** and a **Power BI dashboard**.

---

## Project Highlights

- Built a vendor performance dataset covering **sales, purchases, gross profit, profit margin, inventory efficiency**, and vendor/brand rankings.
- Identified concentration risk using **Pareto (80/20) analysis** for vendor purchase contribution.
- Explored relationships between KPIs using **correlation analysis** and visual diagnostics.
- Produced a **Power BI Vendor Performance Dashboard** to summarize KPIs and spotlight top/low performers.
